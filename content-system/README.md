# 信息与内容工程系统

这里是这套 skill 的主内容系统，也是唯一主库。

当前迁移已经落地，口径如下：

- 旧 skill 名：`wechat-article-writing`
- 新系统名：`information-content-engineering`
- 中文定位：**信息与内容工程系统**

当前仓库路径已经是：

`~/.agents/skills/information-content-engineering`

但系统定位、说明文案、后续对外表述，统一按下面这套理解：

- 这不是单纯“公众号文章写作”工具
- 这是一套覆盖信息处理、内容开发、素材沉淀、选题工程、文章、口播、分镜、归档和方法论沉淀的长期系统
- `content-system/` 是唯一主库
- Obsidian 是查看和编辑界面，不是第二套数据库
- GitHub 是版本管理和同步层

## 四个核心目录

- `01-选题库/`：选题积压、选题规划、待开发线索
- `02-已发布/`：已发布文章、封面与关键发布资产
- `03-素材库/`：素材包、金句、认知表达、案例、钩子、结构资产
- `04-方法论/`：模板、框架、工作流、迭代记录

## 当前迁移规则

1. 新内容优先写入 `content-system/`
2. 不再新建平行的第二套内容库
3. Obsidian 只映射和编辑这套主库
4. Git 提交采用批量整理后再提交，不做每条内容都单独 commit

## 当前收尾范围

当前硬迁移已经完成，剩余工作是收尾：

- 更新所有引用旧路径的脚本、说明和链接
- 清理还保留在说明文件里的历史旧名
- 确认 Obsidian、脚本和外部入口都已切到新路径

这里仍然是唯一主库，按新名字理解，也按新路径运行。
